⚙️ Configuration GitHub Photos

Tes photos seront stockées sur GitHub et accessibles depuis n'importe quel appareil. Suis ces 3 étapes une seule fois :

Étape 1 — Crée un compte GitHub

Va sur github.com/signup et crée un compte gratuit si tu n'en as pas.

Étape 2 — Crée un repository

Vas sur github.com/new, nomme-le mes-recettes, coche "Public" et clique "Create repository". Ensuite dans Settings > Pages, sélectionne main branch comme source.

Étape 3 — Crée un token

Va dans Settings > Developer settings > Personal access tokens > Tokens (classic). Clique "Generate new token", coche repo, copie le token.
